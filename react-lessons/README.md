## 🎯 学習の進め方

1. 各レッスンフォルダの `README.md` を読む
2. サンプルコードを実際に動かして確認
3. 演習問題に取り組む
4. 理解できたら次のレッスンへ

## 🚀 セットアップ

各レッスンフォルダで以下を実行：

```bash
# 依存関係のインストール
npm install

# 開発サーバーの起動
npm start
```

## 💡 Tips

- 公式ドキュメント: https://react.dev
- React Developer Toolsをインストールすると便利
- エラーが出たらブラウザのコンソールを確認
